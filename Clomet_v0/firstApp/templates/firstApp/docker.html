CloMet Docker Image

CloMet can be downloaded locally as a Docker image with some simple steps:

0. Download Docker from this website and follow the tutorial in order to install it.

1. Open a terminal where you want to download the CloMet Docker image.

2. Run the command: docker run --rm -it -p 8000:8000 rogermallol/clomet:latest

You can change the port number 8000 to any unused port in the docker environment.

3. Once the download has finished, open localhost on port 8000 (type localhost:8000 on the browser search bar).

4. To close the application, type CONTROL-C
